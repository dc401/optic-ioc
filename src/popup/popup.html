How It Works

Optic IOC automatically detects and enriches threat indicators on webpages.

Detection

  • Hard Indicators: IPs, domains, URLs, hashes (MD5/SHA1/SHA256), CVEs, emails
  • Yellow Highlights: Indicators detected, enrichment in progress (loading dots)
  • Color-Coded: After enrichment, colors show severity:
    • 🔴 Red = Critical (80-100)
    • 🟠 Orange = High (60-79)
    • 🟡 Yellow = Medium (40-59)
    • 🔵 Blue = Low (20-39)
    • 🟢 Green = Clean (0-19)

Threat Analysis

  • Aggregate Summary: When IOCs are detected, Gemini analyzes all indicators together to provide:
    • Overall risk assessment (critical/high/medium/low)
    • Probable threat actor TTPs
    • Risk and impact to your target organization
    • Recommended actions
  • Location: Blue-bordered panel appears top-right after enrichment completes
  • No Indicators Found? Gemini 2.5 Pro analyzes full page for secrets, payloads, threats
  • Grounding: AI provides research links for validation

Features

  • Hover: See enrichment details (severity, summary, GTI stats)
  • Right-Click: Pivot to VirusTotal, Shodan, OTX, etc. (customizable in Pivots tab)
  • Caching: 30-day cache for faster repeat lookups
  • Private IPs Filtered: RFC1918, link-local addresses skipped
  • Defanged IOCs: Automatically detects and processes hxxp://, [.], [dot], etc.

Debugging & Logs

View detailed logs including Gemini prompts and API calls

Service Worker Console

View background logs, API calls, and Gemini prompts:

  1. Open Chrome Extensions page: chrome://extensions/
  2. Find "Optic IOC"
  3. Click "service worker" link (blue text)
  4. Console opens with all logs

Page Console

View IOC detection and enrichment logs:

  1. Press F12 or right-click → Inspect
  2. Click Console tab
  3. Look for "Optic IOC:" prefixed logs

🔍 Important Log Prefixes

  • [GEMINI] - Prompts sent to Gemini API (Service Worker console)
  • [AGGREGATE] - Threat analysis panel creation (Page console - F12)
  • [GTI] - VirusTotal API calls (Service Worker console)
  • Optic IOC: - General detection and enrichment (Page console)

⚠️ Important Disclaimer

Generative AI Usage Notice

This extension uses Generative AI (Google Gemini or Anthropic Claude via Vertex AI) to:

  • Analyze and summarize threat intelligence data
  • Assess relevance of indicators to your organization
  • Extract TTPs and threat context from webpage content

Important: AI-generated summaries and relevance assessments should be validated by human analysts. The AI may:

  • Misinterpret context or technical details
  • Generate inaccurate or incomplete summaries
  • Make incorrect relevance assessments

Data Privacy: Webpage content and IOCs are sent to Google Vertex AI for processing. Ensure compliance with your organization's data handling policies.

Always verify AI findings with authoritative sources and apply analyst expertise.

Security & Privacy

  • API keys encrypted via Chrome OS keychain (macOS Keychain, Windows DPAPI, Linux libsecret)
  • No data sent to external servers except configured APIs (Gemini, GTI)
  • Cache stored locally with 30-day default TTL
  • Security warnings logged for CTI analyst review

Actions

For internal UKG use only. Report issues to CTI team.
